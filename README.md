# 🌟 Dialogue Generator & Sentence Builder for Tamil Language Learning

> AI-Powered Tamil NLP Platform | Hackathon Project

## 📄 Abstract

An AI-powered web application that helps students learn Tamil grammar interactively. Users input a noun, verb, tense, and gender — the system generates grammatically correct Tamil sentences and real-life dialogues instantly. Built with Python Flask, open-tamil NLP, SQLite, gTTS, and the Claude AI API.

---

## 💡 Why This Project?

- Tamil grammar is complex — no interactive AI tool existed to teach it
- Traditional learning is passive — this makes it active and practical
- Tamil is a classical language that needs modern technology to survive
- Real gap in the market — 80M+ Tamil speakers, zero tools like this

---

## ✨ Features

- ✅ Tamil sentence & dialogue generator (statement, question, conversation)
- ✅ Text-to-Speech — hear Tamil sentences spoken aloud
- ✅ Voice input — speak words via microphone
- ✅ AI sentence variations using Claude API
- ✅ Grammar checker with AI correction
- ✅ Vocabulary browser with search
- ✅ Quiz engine with scoring
- ✅ Points, badges & streak system
- ✅ Save & review sentence history
- ✅ Teacher dashboard & analytics
- ✅ Mobile-installable (PWA)
