5. Building the Grammar Engine
The grammar engine is the heart of the project. It contains the rules for forming Tamil sentences. Create the file backend/grammar_engine.py with the code shown below.

5.1 Understanding Tamil Sentence Structure
Tamil follows a Subject-Object-Verb (SOV) pattern, unlike English which uses Subject-Verb-Object (SVO).

Concept
English (SVO)
Tamil (SOV)
Example
I eat rice
நான் சோறு சாப்பிடுகிறேன்
Pattern
Subject + Verb + Object
Subject + Object + Verb

5.2 grammar_engine.py — Full Code
Copy this entire code block into backend/grammar_engine.py:

# backend/grammar_engine.py
# Tamil Grammar Engine - Rule-based Sentence Generator

class TamilGrammarEngine:

    def __init__(self):
        # Tense suffixes for different genders
        self.present_suffix = {
            'male':   'கிறான்',   # He
            'female': 'கிறாள்',   # She
            'neutral':'கிறது',    # It
            'first':  'கிறேன்',   # I
            'plural': 'கிறார்கள்' # They
        }
        self.past_suffix = {
            'male':   'னான்',
            'female': 'னாள்',
            'neutral':'னது',
            'first':  'னேன்',
            'plural': 'னார்கள்'
        }
        self.future_suffix = {
            'male':   'வான்',
            'female': 'வாள்',
            'neutral':'கும்',
            'first':  'வேன்',
            'plural': 'வார்கள்'
        }
        # Subject pronouns by gender
        self.subjects = {
            'male':   'அவன்',
            'female': 'அவள்',
            'neutral':'அது',
            'first':  'நான்',
            'plural': 'அவர்கள்'
        }

    def get_suffix(self, tense, gender):
        if tense == 'present':
            return self.present_suffix.get(gender, 'கிறான்')
        elif tense == 'past':
            return self.past_suffix.get(gender, 'னான்')
        else:
            return self.future_suffix.get(gender, 'வான்')

    def generate_statement(self, noun, verb, tense, gender):
        subject = self.subjects.get(gender, 'நான்')
        suffix = self.get_suffix(tense, gender)
        sentence = f'{subject} {noun} {verb}{suffix}'
        return sentence

    def generate_question(self, noun, verb, tense, gender):
        statement = self.generate_statement(noun, verb, tense, gender)
        # Remove last char and add question suffix
        question = statement[:-1] + 'ஆ?'
        return question

    def generate_dialogue(self, noun, verb, tense, gender):
        stmt = self.generate_statement(noun, verb, tense, gender)
        qstn = self.generate_question(noun, verb, tense, gender)
        dialogue = [
            {'speaker': 'நண்பன்', 'line': qstn},
            {'speaker': 'நான்',   'line': stmt},
        ]
        return dialogue

6. Building the Flask Backend (app.py)
Create backend/app.py with the following code. This file runs your web server and handles requests from the browser.

# backend/app.py
from flask import Flask, request, jsonify, render_template
from flask_cors import CORS
import sys, os

sys.path.insert(0, os.path.dirname(__file__))
from grammar_engine import TamilGrammarEngine

app = Flask(
    __name__,
    template_folder='../frontend/templates',
    static_folder='../frontend/static'
)
CORS(app)
engine = TamilGrammarEngine()

@app.route('/')
def index():
    return render_template('index.html')

@app.route('/generate', methods=['POST'])
def generate():
    data = request.get_json()
    noun       = data.get('noun', '')
    verb       = data.get('verb', '')
    tense      = data.get('tense', 'present')
    gender     = data.get('gender', 'first')
    sent_type  = data.get('sentence_type', 'statement')

    if sent_type == 'statement':
        result = engine.generate_statement(noun, verb, tense, gender)
        return jsonify({'type': 'statement', 'output': result})
    elif sent_type == 'question':
        result = engine.generate_question(noun, verb, tense, gender)
        return jsonify({'type': 'question', 'output': result})
    else:
        result = engine.generate_dialogue(noun, verb, tense, gender)
        return jsonify({'type': 'dialogue', 'output': result})

if __name__ == '__main__':
    app.run(debug=True, port=5000)

9. JavaScript Logic (main.js)
Create frontend/static/js/main.js:

async function generateSentence() {
  const noun         = document.getElementById('noun').value.trim();
  const verb         = document.getElementById('verb').value.trim();
  const tense        = document.getElementById('tense').value;
  const gender       = document.getElementById('gender').value;
  const sentence_type = document.getElementById('sentence_type').value;

  if (!noun || !verb) {
    document.getElementById('result').innerHTML = 'Please enter both noun and verb.';
    return;
  }

  const response = await fetch('/generate', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ noun, verb, tense, gender, sentence_type })
  });
  const data = await response.json();

  if (data.type === 'dialogue') {
    let html = '<strong>உரையாடல்:</strong><br>';
    data.output.forEach(d => {
      html += `<b>${d.speaker}:</b> ${d.line}<br>`;
    });
    document.getElementById('result').innerHTML = html;
  } else {
    document.getElementById('result').textContent = data.output;
  }
}

Solution

Run: pip install flask inside your venv

Ensure your file is saved as UTF-8 in VS Code
